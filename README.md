# Web Interface for UnitCommitment.jl

This repository contains a web interface for [UnitCommitment.jl][1], an optimization package for the
Security-Constrained Unit Commitment Problem (SCUC). The interface allows users to
visualize, modify and solve test cases directly from the browser, without requiring
software download or programming skills.

[1]: https://github.com/ANL-CEEESA/UnitCommitment.jl
